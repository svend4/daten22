# Финансы и деньги по типам MBTI

## Отношение к деньгам по типам

**NT** — Стратегический подход
- INTJ: Долгосрочные инвестиции, пассивный доход
- INTP: Исследуют финансовые системы, часто импульсивны
- ENTJ: Активные инвестиции, рост капитала
- ENTP: Рискованные инвестиции, множество проектов

**NF** — Ценностный подход  
- INFJ: Этичные инвестиции, баланс
- INFP: Деньги для свободы, не цель
- ENFJ: Для помощи другим
- ENFP: Импульсивны, мечты > планирование

**SJ** — Консервативный подход
- ISTJ: Сбережения, бюджет, дисциплина
- ISFJ: Безопасность семьи
- ESTJ: Эффективное управление
- ESFJ: Для семьи и общества

**SP** — Практический подход
- ISTP: Минималисты, свобода
- ISFP: Деньги для красоты, опыта
- ESTP: Риск, быстрые деньги
- ESFP: Живут сейчас, импульсивны

## Инвестиционные стратегии

**Консервативные:** ISTJ, ISFJ, ESTJ
**Сбалансированные:** INTJ, ENTJ, INFJ
**Агрессивные:** ENTP, ESTP, ENTJ
**Избегают:** INFP, ISFP, ENFP (часто)
