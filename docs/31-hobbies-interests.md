# Хобби и интересы по типам MBTI

## Типичные увлечения

**NT**
- INTJ: Стратегические игры, программирование, чтение
- INTP: Программирование, философия, науки
- ENTJ: Бизнес-проекты, шахматы, спорт
- ENTP: Дебаты, множество проектов, исследования

**NF**
- INFJ: Писательство, психология, искусство
- INFP: Творчество, музыка, литература
- ENFJ: Волонтерство, театр, коучинг
- ENFP: Творчество, социальные события, искусство

**SJ**
- ISTJ: Коллекционирование, садоводство, история
- ISFJ: Рукоделие, готовка, семейные альбомы
- ESTJ: Спорт, общественные организации, гольф
- ESFJ: Социальные события, готовка, волонтерство

**SP**
- ISTP: Механика, экстремальный спорт, мотоциклы
- ISFP: Искусство, музыка, природа
- ESTP: Спорт, приключения, автомобили
- ESFP: Танцы, вечеринки, shopping, travel
