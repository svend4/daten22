## Рекомендации по карьере

### Для интровертов (I)

**Ищите:**
- Возможность работать независимо
- Глубокую концентрацию
- Минимум отвлечений
- Контроль над средой

**Избегайте:**
- Открытых офисов
- Постоянных собраний
- Холодных звонков
- Чрезмерной социализации

---

### Для экстравертов (E)

**Ищите:**
- Работу в команде
- Взаимодействие с людьми
- Динамичную среду
- Обратную связь

**Избегайте:**
- Изолированной работы
- Монотонности
- Отсутствия общения
- Одиночных проектов

---

### Для сенсориков (S)

**Ищите:**
- Практическое применение
- Конкретные результаты
- Чёткие инструкции
- Традиционные методы

**Избегайте:**
- Чисто теоретической работы
- Неопределённости
- Постоянных изменений
- Абстрактных концепций

---

### Для интуитов (N)

**Ищите:**
- Креативность
- Стратегическое мышление
- Возможности для инноваций
- Концептуальную работу

**Избегайте:**
- Рутины
- Детальной работы
- Отсутствия видения
- Микроменеджмента

---

### Для мыслителей (T)

**Ищите:**
- Логические задачи
- Объективные критерии
- Меритократию
- Техническую экспертизу

**Избегайте:**
- Политики офиса
- Эмоциональных манипуляций
- Субъективных оценок
- Необъективности

---

### Для чувствующих (F)

**Ищите:**
- Осмысленную работу
- Помощь людям
- Гармоничную среду
- Признание вклада

**Избегайте:**
- Бесчувственной среды
- Конфликтов
- Холодной логики
- Изоляции

---

### Для решающих (J)

**Ищите:**
- Структуру
- Чёткие дедлайны
- Планирование
- Организацию

**Избегайте:**
- Хаоса
- Неопределённости
- Постоянных изменений
- Отсутствия плана

---

### Для воспринимающих (P)

**Ищите:**
- Гибкость
- Разнообразие
- Спонтанность
- Адаптивность

**Избегайте:**
- Жёстких графиков
- Микроменеджмента
- Рутины
- Строгой иерархии
