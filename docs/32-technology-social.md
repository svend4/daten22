# Технологии и социальные сети по типам MBTI

## Использование интернета

**NT** — Early adopters
- INTJ: Reddit, специализированные форумы
- INTP: Programming communities, Discord
- ENTJ: LinkedIn, профессиональные сети
- ENTP: Twitter (дебаты), Reddit

**NF** — Креативное использование
- INFJ: Блоги, Instagram (эстетика)
- INFP: Tumblr, Pinterest, творческие платформы
- ENFJ: Facebook, Instagram (связи)
- ENFP: Instagram, TikTok, множество платформ

**SJ** — Консервативное
- ISTJ: Email, LinkedIn
- ISFJ: Facebook (семья)
- ESTJ: LinkedIn, профессиональные
- ESFJ: Facebook, Instagram

**SP** — Активное
- ISTP: YouTube (DIY), Reddit
- ISFP: Instagram, Pinterest
- ESTP: Snapchat, Instagram Stories
- ESFP: Instagram, TikTok, все популярное

## Gaming

**Высокая вовлеченность:** INTP, INTJ, ISTP, ENTP
**Средняя:** INFP, ENFP, ESTP
**Низкая:** SJ типы, ENFJ
