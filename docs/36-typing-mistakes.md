# Ошибки типирования

## Частые путаницы

**INTJ vs INTP**
- INTJ: Ni-Te (видение + эффективность)
- INTP: Ti-Ne (логика + возможности)
*Ключ:* INTJ решительны, INTP исследуют

**INFJ vs INFP**
- INFJ: Ni-Fe (видение + гармония)
- INFP: Fi-Ne (ценности + возможности)
*Ключ:* INFJ для других, INFP для себя

**ENTJ vs ESTJ**
- ENTJ: Intuitive, стратеги
- ESTJ: Sensor, традиционалисты
*Ключ:* ENTJ будущее, ESTJ прошлое

**ENFP vs ESFP**
- ENFP: Ne-Fi (возможности + ценности)
- ESFP: Se-Fi (опыт + ценности)
*Ключ:* ENFP абстрактны, ESFP конкретны

**ISTJ vs ISTP**
- ISTJ: Si-Te (память + логика)
- ISTP: Ti-Se (логика + действие)
*Ключ:* ISTJ планируют, ISTP импровизируют

**INFP vs ISFP**
- INFP: Intuitive, идеалисты
- ISFP: Sensor, эстеты
*Ключ:* INFP идеи, ISFP опыт

## Признаки ошибок

❌ Типирование по поведению, не функциям
❌ Стереотипы вместо анализа
❌ Социальные роли как тип
❌ Желаемое vs реальное
❌ Игнорирование развития
