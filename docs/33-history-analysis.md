# Типы личности в истории

## Исторические периоды и типы

**Древний мир**
- Философы: INTP, INTJ (Сократ, Платон)
- Завоеватели: ENTJ, ESTP (Александр Македонский)
- Лидеры: ENFJ (Будда)

**Средневековье**
- Преобладание: SJ типов
- Рыцари: ESTJ, ISTJ
- Монахи: ISFJ, INFJ

**Возрождение**
- Расцвет: NT и NF типов
- Художники: ISFP, INFP (Микеланджело)
- Изобретатели: ENTP, INTP (Леонардо да Винчи)

**Просвещение**
- Философы: INTP, INTJ
- Революционеры: ENTP, ENTJ

**Индустриальная эпоха**
- Промышленники: ENTJ, ESTJ
- Изобретатели: INTP, ENTP

**Современность**
- Разнообразие всех типов
- Tech: NT типы
- Social movements: NF типы
